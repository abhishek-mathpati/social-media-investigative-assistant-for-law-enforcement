## No Platform Bypass or Exploitation

SMIA does **not**:
- Bypass social media platform security mechanisms
- Exploit private or undocumented APIs
- Perform credential harvesting
- Circumvent access controls
- Extract backend databases or hidden metadata

All interactions occur through **standard user interfaces** using assisted navigation.

---

## Evidence Collection Methodology

- The system captures **visual evidence only** (screenshots)
- No raw text scraping or direct DOM/database extraction is performed
- Evidence is compiled into **timestamped PDF reports**
- This approach preserves visual authenticity and supports auditability
